# Training Program: Codex, Eclipse, Java, Text Files, and Postgres

This workspace contains a complete hands-on training program for showing developers how to use Codex while working in Eclipse on a Java data-loading application.

## Audience

- Beginning to intermediate Java developers
- Developers new to AI-assisted coding with Codex
- Teams learning file ingestion, validation, JDBC, and PostgreSQL basics

## Learning Outcomes

By the end of the training, learners can:

- Use Codex from an Eclipse-centered workflow.
- Ask Codex for explanations, implementation help, refactoring, and tests.
- Import and run a Java Maven project in Eclipse.
- Read structured data from a text file.
- Validate and map file records into Java objects.
- Load valid records into a PostgreSQL table using JDBC.
- Use Codex to diagnose errors and improve code quality.

## Quick Start

1. Install Java 17 or newer.
2. Install Eclipse IDE for Enterprise Java and Web Developers.
3. Install PostgreSQL locally, or use the included Docker Compose file.
4. Install Maven, or use Eclipse's built-in Maven project support.
5. Import `java-file-to-postgres` into Eclipse as an existing Maven project.
6. Follow `docs/learner-handout.md`.
